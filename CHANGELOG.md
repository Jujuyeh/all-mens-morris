# Changelog

## v1.0.0 - 2026-06-22

Initial playable release.

- Play classic Nine Men's Morris and several custom board profiles.
- Select Standard, Long Morris, and Lesker Morris rulesets from the main menu.
- Play local two-player, CPU Easy, CPU Hard, or FX-C link-cable matches.
- Use mandatory capture, movement, flying, material wins, block wins, and draw
  detection.
- Includes animated menus, attract mode, compact bossa-style menu music,
  sound effects, RGB mill feedback, undo/redo history, and a compact side HUD.
- Includes TableTop Studio for editing board/rule profiles, global sprites, and
  menu audio source data.

Known limitations:

- FX-C link multiplayer expects two units running the same game build.
- Advanced link desync recovery is out of scope for this release.
- No persistent save/settings storage yet.
- The FX-C build is close to the ATmega32U4 flash limit.
